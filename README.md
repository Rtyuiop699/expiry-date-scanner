## 🧠 Expiry Date Scanner App

تطبيق أندرويد لإدارة المنتجات باستخدام الباركود، يتيح للمستخدم مسح الباركود عبر الكاميرا والبحث عن المنتج محلياً أو عبر الإنترنت، ثم حفظه في قاعدة بيانات محلية مع إمكانية تعديل البيانات مثل الاسم وتاريخ الانتهاء والصورة.

### ✨ المميزات
- 📷 مسح الباركود باستخدام الكاميرا
- 🌍 البحث عن المنتج عبر الإنترنت (API)
- 💾 حفظ المنتجات في قاعدة بيانات محلية
- ✏️ تعديل بيانات المنتجات بسهولة
- 📋 عرض قائمة المنتجات بشكل منظم
- 🔍 واجهة بسيطة وسهلة الاستخدام

### ⚙️ التقنيات المستخدمة
- Kotlin (Android)
- SQLite Database
- Retrofit API
- ZXing Barcode Scanner
- RecyclerView
